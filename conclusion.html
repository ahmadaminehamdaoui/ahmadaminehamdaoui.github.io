Conclusion

Les trois moteurs de jeu présentés précédemment possèdent donc tous des points forts et des points faibles. Nous énumérerons les avantages et les inconvénients de chaque moteur de jeu, et enfin, nous comparerons Unity et Unreal (car Ursina n'est pas du tout le même genre de moteur de jeu).

Unity Engine

Les points forts:

  • Facile à prendre en main, pratique pour les débutants ;
  • Flexible, permet de créer n'importe quell type de jeu ;
  • Gratuit dans la plupart des cas ;

Les points faibles:

  • Possède moins d'outils que son concurrent, ils sont parfois instables ;
  • L'utilisation d'applications tierces de l'asset store est souvent nécessaire ;
  • Les graphismes doivent être réajustés pour obtenir un beau résultat ;

Unreal Engine

Les points forts:

  • Des graphismes superbes intégrés au moteur de jeu ;
  • Facilite la création des jeux de type FPS ;
  • Possède une grande variété d'outils fonctionnels ;

Les points faibles:

  • Plus difficile à prendre en main (le C++ est difficile à apprendre) ;
  • Une interface qui se remplit rapidement ;
  • Unreal prend une part plus importantes des revenus (5% au dessus de 5000$) ;

Ursina Engine

Les points forts:

  • Un langage facile à prendre en main (Python) ;
  • Un moteur de jeu rapide ;
  • Une bonne documentation ;

Les points faibles:

  • Des outils limités ;
  • Possède une petite communauté ;
